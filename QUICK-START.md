## 🚀 What You Need To Do:

### 1️⃣ Create Storyblok Components (45 mins)
Open: **[MANUAL-SETUP.md](./MANUAL-SETUP.md)**

Follow the step-by-step guide to create all 24 components in Storyblok.

**Tip:** Start with the simpler ones (nav_item, trust_badge) to get familiar with the interface.

---

### 2️⃣ Create "home" Story & Add Content (30 mins)
Open: **[COMPLETE-CONTENT-GUIDE.md](./COMPLETE-CONTENT-GUIDE.md)**

1. Create a new story called "home" in Storyblok
2. Add all 13 sections in order
3. Copy-paste content from the guide

**Tip:** Use Ctrl+F to quickly find the content you need for each section.

---

### 3️⃣ Upload Image & Publish (5 mins)

1. Upload `public/hero-bg.jpg` to Storyblok assets
2. Set it as the background in hero_section
3. Click **Publish**
4. Your site is LIVE! 🎉
